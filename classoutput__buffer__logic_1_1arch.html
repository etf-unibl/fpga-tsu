Detailed Description

Architecture of output_buffer_logic.

When the first element is written into the fifo buffer it gets popped automatically through this logic Every subsequent reading will pop the next element from the fifo buffer There is an asynchronous reset, and also the logic is reset when the fifo buffer gets empty
