Detailed Description

Entity that describes the register map There are seven register. sys_time register for storing Unix time, status for status information and interrupt sysbsystem, control for some control flags, and 4 register for storing information about transition of input signal, fall_ts_* when falling edge, and rise_ts_* when rising edge. Sufix h or l denotes if unix time or nanoseconds is store in register
