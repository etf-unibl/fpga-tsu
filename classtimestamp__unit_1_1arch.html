Detailed Description

Architecture that does functional description of design.

Design contains fifo buffers, interrupt logic, custom detection_logic module, and reg_map module. Inside detection_logic there is a counter module and based on that counting, detection_logic gives its outputs that are then being stored into FIFO buffers through the reg_map
