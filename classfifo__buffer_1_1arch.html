◆ increment_head_tail()

increment_head_tail ( clk_i ,
rst_i  )

Process for incrementing the head and tail of the buffer Head gets incremented if write enable signal is '1' and the buffer is not full Old values get overwritten if write enable signal is '1' and the buffer is full Tail gets incremented when the read enable signal is '1' and the buffer is not empty If the read enable signal is not active, or if the buffer is empty, the read valid signal is not active When an element gets popped from the buffer successfully the read valid signal is active
