Detailed Description

This is a synchronous design that, as a port, has one-bit input on which digital waveform is brought. There are two output ports. One is 32-bit output that fills timestamp_value register with the value of on the input (0 or 1). The other one is set to one when there is a change on the input that needs to be recognized (0->1 or 1->0).
