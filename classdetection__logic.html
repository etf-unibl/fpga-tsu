Detailed Description

Entity that describes the inputs and output of the detection_logic unit There is an input data_i single-bit signal which transitions are gonna be detected by this design. Each time transition is detected, 4 outputs fall_tses & rise_tses are gonna be updated according to the exact time when it's happened (both Unix time moment and nanosec time moment). Additionally, there is an input unix_start_value_i as init value for the Unix counter and output unix_time_o as output of the same counter.
