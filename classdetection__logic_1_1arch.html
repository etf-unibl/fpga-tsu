Detailed Description

Architecture with functional definition of detection_logic.

Architecture is a simple design, where on each clock, based on the current value of the input and the previous one, output 'we' is set to 1 if there is a transition on the input, and to 0 if there is not. The other output is constantly updated based on the current value of the input.
